Gramática

La gramática para las expresiones regulares es G = { V, E, R, S} donde V = {A}, el conjunto de variables, E = {a, ..., z, A, ..., Z, 0, ..., 9} el conjunto de terminales, R el conjunto de producciones y S = A la variable inicial.

Las producciones son las siguientes:

A -> (A)*A

A -> (A)+A

A -> (A)A

A -> (A)*|A

A -> (A)+|A

A -> (A)|A

A -> (A)*

A -> (A)+

A -> (A)

A -> #*A

A -> #+A

A -> #A

A -> #*|A

A -> #+|A

A -> #|A

A -> #*

A -> #+

A -> #

Donde # representa un elemnto de E.
